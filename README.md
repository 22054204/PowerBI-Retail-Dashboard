# 📊 Retail Sales Analysis Dashboard

An end-to-end Business Intelligence project developed using **Power BI**, **SQL**, and **Excel** to analyze retail sales data and generate meaningful business insights. The dashboard helps visualize sales performance, customer demographics, product category trends, and purchasing behavior through interactive reports.

---

## 📌 Project Overview

This project focuses on analyzing retail transaction data to identify business trends and support data-driven decision-making.

The workflow includes:

- Data collection from an Excel dataset
- Data preparation using SQL
- Data visualization in Power BI
- Business insight generation using interactive dashboards

---

## 🛠️ Technologies Used

- **Power BI Desktop**
- **SQL**
- **Microsoft Excel**

---

## 📂 Dataset

The dataset contains retail transaction records including:

- Transaction ID
- Customer ID
- Date
- Age
- Gender
- Product Category
- Quantity Purchased
- Price per Unit
- Total Amount

---

## 📈 Dashboard Features

The dashboard provides interactive visualizations for:

- 📅 Monthly Revenue Trend
- 🛍️ Revenue by Product Category
- 👥 Customer Revenue by Gender
- 🎯 Revenue by Customer Age
- 📦 Quantity vs Revenue Analysis
- 🔍 Interactive Filters by Category, Date, and Gender

---

## 💡 Business Insights

Some key insights generated from the dashboard include:

- Identification of monthly sales trends and seasonal demand
- Comparison of revenue across different product categories
- Analysis of customer purchasing behavior based on age and gender
- Understanding the relationship between product quantity and revenue
- Interactive filtering to explore sales data from different perspectives

## 🚀 How to Run

1. Clone or download this repository.
2. Open `Retail_Sales_Dashboard.pbix` using **Power BI Desktop**.
3. If required, reconnect the Excel dataset.
4. Explore the dashboard using the available slicers and filters.
