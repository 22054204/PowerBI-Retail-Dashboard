# 📖 Overview

This project demonstrates how Business Intelligence tools can be used to analyze retail sales data and transform raw transactional records into meaningful business insights.

The dashboard enables users to monitor sales performance, analyze customer demographics, compare product categories, and identify purchasing trends using interactive visualizations.

---

# 🚀 Project Objectives

- Analyze retail sales performance
- Identify monthly sales trends
- Understand customer purchasing behavior
- Compare product category performance
- Visualize revenue using interactive dashboards
- Support data-driven business decisions

---

| Tool | Purpose |

# 📂 Dataset

The dataset contains more than **1000 retail transactions** with the following information:

- Transaction ID
- Customer ID
- Purchase Date
- Customer Age
- Gender
- Product Category
- Quantity Purchased
- Price Per Unit
- Total Amount

---

# 📊 Dashboard Features

The dashboard provides interactive analysis through:

### 📈 Monthly Revenue Trend

Visualizes monthly sales performance to identify seasonal demand and business trends.

---

### 🛍 Revenue by Product Category

Compares revenue generated by different product categories to identify best-selling products.

---

### 👥 Customer Revenue by Gender

Displays sales distribution between male and female customers.

---

### 🎯 Revenue by Customer Age

Analyzes purchasing patterns across different age groups.

---

### 📦 Quantity vs Revenue Analysis

Shows the relationship between quantity sold and generated revenue.

---

### 🎛 Interactive Filters

The dashboard supports dynamic filtering using:

- Product Category
- Gender
- Transaction Date

allowing users to explore the data from multiple perspectives.

1. Clone or download the repository.
2. Open **Retail_Sales_Dashboard.pbix** in Power BI Desktop.
3. If prompted, reconnect the Excel dataset.
4. Explore the dashboard using the available slicers and filters.
